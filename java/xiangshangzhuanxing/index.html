Java向上转型详解

关键词： 向上转型 , 同名变量 ，同名方法

       对象的上转型其实就是多态的一种表现。不知道大家有没有遇到这种情况！向上转型的对象访问的变量是父类原来的而不是子类重写的，但访问的方法是子类重写的！
       大家都知道当向上转型的对象访问的方法是子类重写的方法（如果子类重写了父类的方法）。但为什么子类重写了父类的变量，向上转型的对象却访问的是父类的对象呢？今天我将和大家一起分析这个问题！
